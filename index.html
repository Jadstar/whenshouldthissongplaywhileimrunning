Pick a song to run with

How does it work?

Spotify has a bunch of characteristics for each song which can be accessed via their Web API.

E.g.

  • Danceability
  • Tempo
  • Acousticness
  • Instrumentalness
  • Etc.

By going through songs and theorising when they should play manually, I can separate these songs into 5 different running stages.

  • Warmup
  • Recovery/ Base Run
  • Tempo Run
  • Race Run
  • Cool down
  • Falloff

I've learnt that there is a lot of overlap in which songs are viable for the sections, (race and tempo are practically interchangeable, as are recovery and cooldown). So the priority is to maintain the 'momentum' of the run via the songs playing, rather than guarantee that a specific song is only played during an exact stage of running.

Essentially, it's just about weeding out 'run-destroying' songs in a playlist, and trying to time motivational music when it is needed the most.

Since I can't test every song in the world, and I have my own personal biases, I made this site so others can agree/disagree with me so I can try and improve this system.
